This is React based component

## Converting Bitcoin

 -> converts the inputed bitcoin amount to other currencies.<br />
 -> uses data from API and updates every 1 min.<br />
 -> can see the rates in up to three currencies.<br />
 -> you can delete/reselect a currency<br />

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.

### `npm run storybook`

Runs a storybook with converter componenet <br />
Open [http://localhost:6006](http://localhost:6006) to view it in the browser.
